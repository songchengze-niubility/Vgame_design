# Encounter Pacing Guide

## 节奏设计原则

Vgame 是跑酷战斗游戏，关卡节奏的核心是**"跑"和"打"的交替**：

- 纯跑酷太久 → 无聊，失去战斗快感
- 纯战斗太久 → 疲劳，失去跑酷乐趣
- 最佳节奏 → 30-60秒战斗 + 10-20秒跑酷/喘息，循环往复

## 各玩法的节奏模型

### 主线关卡

```
典型时长: 60-150秒
结构: [引入10s] → [小怪段30s] → [跑酷10s] → [精英段20s] → [跑酷10s] → [Boss段30-60s]
特点: 从简到难递进，Boss在最后
```

| 章节 | 总时长预期 | Loop数 | 怪物密度 |
|---|---|---|---|
| Ch0（序章） | 30-60s | 1-2 | 低，教学为主 |
| Ch1 | 60-90s | 2-3 | 中等 |
| Ch2 | 90-120s | 3 | 中高 |
| Ch3 | 120-150s | 3-4 | 高 |
| Ch4 | 120-150s | 3-4 | 高 |

### 资源副本（金币/经验/专武）

```
典型时长: 60-90秒
结构: [小怪群30s] → [小怪群30s] → [精英/小Boss 30s]
特点: 高密度、高效率，追求"刷"的爽感
```

### 旅券副本

```
典型时长: 60-120秒
结构: [跑酷5s] → [Boss战60-120s]
特点: 快速进入Boss战，考验单体输出和机制理解
```

### 无限挑战

```
典型时长: 无限递增
结构: 每轮 [普通怪] → [精英怪] → [Boss]，3关一循环
特点: 难度递增，越往后越耗时
怪物等级公式: level = 27 + 1000*(n-1)/((n-1)+38)
```

### 爬塔

```
典型时长: 每层30-60秒
结构: 层类型决定内容（小层/中层/大层）
特点: 每层独立，一次性+周期刷新混合
```

### 竞技场(PVPVE)

```
典型时长: 90-120秒
结构: 双方共享刷怪，未清怪攻击基地
特点: PVP压力下的PVE效率比拼
```

### 大秘境

```
典型时长: 120-180秒
结构: 环境词缀 + 递增怪物等级 + 多Loop
特点: 后期验证，需要box宽度和环境适应
```

## 刷怪密度参考

### 每 Loop 击杀数设计

| 难度感 | OrdinaryKill 数 | 战斗时长预期 | 适用场景 |
|---|---|---|---|
| 轻松 | 3-5 | 10-15s | 教学、过渡段 |
| 标准 | 5-8 | 15-25s | 主线中段 |
| 紧张 | 8-12 | 25-40s | 主线后段、资源本 |
| 高压 | 12-20 | 40-60s | 挑战模式、高难 |

### 精英/Boss 配置

| 怪物类型 | 每 Loop 数量 | 战斗时长贡献 |
|---|---|---|
| 普通怪 | 5-20 | 每只 2-5s |
| 精英怪 | 1-3 | 每只 10-20s |
| Boss | 1 | 30-120s |

## 刷怪位置设计

### X 坐标规则

实体按 X 坐标定位，镜头滚到对应位置时加载：

| 位置策略 | 说明 | 适用 |
|---|---|---|
| 前方刷出 | 怪物在玩家前方出现 | 大部分场景 |
| 两侧夹击 | 前后同时出怪 | 增加难度 |
| 高台射击 | 远程怪在高处平台 | 增加空间层次 |
| 追击型 | 怪物从后方追来 | 增加紧张感 |

### Loop 内的刷怪节奏

Loop 内怪物不是一次全出，而是分批：

```
Loop 开始
  → 第1批: 2-3只普通怪（热身）
  → 间隔2-3秒
  → 第2批: 3-5只普通怪（主力）
  → 间隔2-3秒
  → 第3批: 1精英 + 2普通（高潮）
  → 击杀满足 → Loop 结束
```

## 难度调节手段

### Excel 层可调（不改场景）

| 手段 | 字段 | 效果 |
|---|---|---|
| 怪物血量系数 | level.HpCoefficient | 全关怪物HP统一缩放 |
| 怪物攻击系数 | level.AttackCoefficient | 全关怪物ATK统一缩放 |
| 怪物防御系数 | level.ArmorCoefficient | 全关怪物DEF统一缩放 |
| 推荐等级 | UIlevel.RecommendLevel | UI提示 |
| 怪物等级映射 | 战斗框架玩法难度线 | 全玩法等级曲线 |
| Boss战时间 | BossBattleTime | 超时即失败 |

### 场景层可调（需要关卡编辑器）

| 手段 | 效果 |
|---|---|
| 增减怪物数量 | 改变Loop击杀条件 |
| 调整刷怪间隔 | 改变战斗密度 |
| 增减地形间距 | 改变跑酷难度 |
| 添加/移除移动平台 | 改变空间复杂度 |
| 调整Loop长度 | 改变循环节奏 |
| 增加环境机关 | 增加额外压力 |

## 常见问题排查

| 问题 | 排查方向 |
|---|---|
| 玩家觉得太简单 | HP/ATK系数太低？怪物数量太少？Loop条件太容易？ |
| 玩家觉得太难 | HP/ATK系数太高？怪物等级映射跳跃太大？存活时间不够？ |
| 关卡太短/太长 | Loop数量？击杀条件数？跑酷段长度？ |
| 节奏单调 | 缺少跑酷/战斗交替？缺少精英/Boss节点？ |
| Boss不够有压力 | Boss HP系数？战斗时间限制？阶段机制？ |
| 跑酷段无聊 | 缺少平台跳跃？缺少道具？缺少视觉变化？ |
