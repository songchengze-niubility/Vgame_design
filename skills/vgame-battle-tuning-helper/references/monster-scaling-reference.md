# Monster Scaling Reference

## 数据来源

Sheet "怪物标准模型", Row 12-511, 共500级。

## Excel 坐标

| 列 | 字段 | 用途 |
|---|---|---|
| B | 等级 | 怪物等级 1-500 |
| Z | 生命(最终) | 基准怪物最终HP |
| AA | 攻击(最终) | 基准怪物最终ATK |
| AB | 防御(最终) | 基准怪物最终DEF |
| AC | 减伤率 | 基于防御计算的减伤 |
| AD | ehp | 有效生命 |
| AF | TTK(秒) | 标准人击杀时间 |
| AG | 人数系数 | 队伍人数修正 |
| AH | 攻击系数修正 | 怪物ATK修正 |
| AI | BOSS生命系数修正 | Boss HP乘数 |
| AJ | 小怪生命系数修正 | 小怪HP乘数 |
| AK | 小怪攻击系数修正 | 小怪ATK乘数 |
| AL | BOSS生命 | Boss最终HP |
| AM | BOSS攻击 | Boss最终ATK |
| AN | BOSS防御 | Boss最终DEF |
| AO | 小怪生命 | 小怪最终HP |
| AP | 小怪攻击 | 小怪最终ATK |

## 怪物类型与系数

| 类型 | HP系数来源 | ATK系数来源 | 典型TTK |
|---|---|---|---|
| 普通怪 | AJ(小怪生命系数) | AK(小怪攻击系数) | 短 |
| 盾怪 | 普通×2~3 | 普通×0.8 | 中 |
| 精英怪 | 普通×3~5 | 普通×1.2~1.5 | 中-长 |
| Boss | AI(BOSS生命系数) | AH(攻击系数修正) | 长 |

## 属性成长曲线

怪物属性从 Level 1 → 500 使用指数曲线（曲率 0.999999）:

| 属性 | Level 1 | Level 50 | Level 100 | Level 200 | Level 500 |
|---|---|---|---|---|---|
| 裸体HP | 79 | ~2,000 | ~5,000 | ~15,000 | 39,500 |
| 裸体ATK | 23.7 | ~600 | ~1,500 | ~4,500 | 11,850 |
| 裸体DEF | 15.8 | ~400 | ~1,000 | ~3,000 | 7,900 |

（精确值从 Excel 读取，此处为趋势参考）

## 自动读取接口

脚本从 Excel Row `11 + level` 读取对应等级数据：
- `row = 11 + monster_level`
- 如 Level 50 → Row 62
