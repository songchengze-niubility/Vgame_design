# Combat Math Reference

## 伤害公式

### 基础伤害

```
防御有效值 = 受击者防御 × (1 - 攻击者无视防御)
基础伤害 = 伤害基数 × [1 - 防御有效值 / (防御有效值 + 24000)]
基础伤害保底 = 伤害基数 × 5%
```

防御常数 = **24000**

### 暴击

```
暴击后伤害 = 基础伤害 × 暴击伤害倍率
```

暴击率上限1, 下限0。

### 格挡

```
格挡概率 = 受击者格挡率 - 攻击者破击率
格挡后伤害 = 暴击后伤害 × (1 - 0.5 - 格挡强度)
```

基础格挡减伤 = **50%**

### 最终伤害

```
最终伤害 = 格挡/暴击后伤害
  × (1 + 攻击者伤害加成)
  × (1 - 受击者伤害减免)       [上限75%]
  × (1 + 受击者受伤增加)
  × (1 + 攻击者玩法伤害加成)
  × (1 - 受击者玩法伤害减免)
```

最终伤害保底 = 1

### 命中判定

```
命中概率 = 攻击者命中 - 受击者闪避
保底命中率 = 10%
```

## DPS 计算模型

### 标准人攻击节奏

| 攻击类型 | 每秒次数 | 说明 |
|---|---|---|
| 普攻 | 1.0 /s | 基础攻速 |
| 必杀 | 0.55 /s | 能量充满后释放 |
| 技能 | 0.4 /s | 冷却后释放 |
| **合计** | **1.95 /s** | 总输出频率 |

### DPS 公式

```
单次伤害 = ATK × 技能系数 × [1 - DEF/(DEF+24000)]
期望伤害 = 单次伤害 × (1 + (暴击率 × (暴击伤害-1))) × (1 + 增伤乘区)
DPS = 期望伤害 × 攻击频率
```

简化为：
```
DPS ≈ ATK × 系数均值 × 防御穿透率 × 暴击期望 × 增伤期望 × 攻速
```

### TTK（击杀时间）

```
TTK = 怪物有效HP / 玩家DPS
有效HP = 怪物HP / (1 - 怪物减伤率)
```

### 玩家存活时间

```
存活时间 = 玩家有效HP / 怪物DPS
玩家有效HP = 玩家HP / (1 - 玩家减伤率)
```

## 难度判断标准

| TTK vs 预期时长 | 难度评价 |
|---|---|
| TTK < 预期×0.5 | 过于简单（碾压） |
| TTK = 预期×0.5~0.8 | 偏简单 |
| TTK = 预期×0.8~1.2 | 适中 |
| TTK = 预期×1.2~1.5 | 偏难 |
| TTK > 预期×1.5 | 过难（可能超时/卡关） |
| 存活时间 < TTK | 打不过（会被打死） |
