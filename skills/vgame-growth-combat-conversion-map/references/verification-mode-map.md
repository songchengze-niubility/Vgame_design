# 正式玩法验证地图

## 验证层级

| 玩法 | 验证重点 | 关联成长 |
|---|---|---|
| 主线 | 综合账号准备度、教学和基础成长闭环。 | 角色等级、突破、早期星盘、阵容扩充、基础装备。 |
| 资源副本 | 日常供给效率和基础战斗稳定性。 | 基础 DPS、生存、扫荡门槛。 |
| 旅券副本 | 中期 boss/节奏处理和稳定装备成长获取。 | 角色机制、专武、旅券需求、boss 输出窗口。 |
| 映射装备副本 | 后期短循环 farm 效率和终局装备获取。 | 映射装备、爆发效率、清怪效率、掉落追求。 |
| 无限挑战 | 长线 endurance 和综合账号深度。 | 队伍续航、输出稳定性、生存、成长广度。 |
| 竞技场（PVPVE） | 竞争效率、漏怪惩罚、基地生存和赛季留存。 | 输出效率、生存、炮台/基地相关成长、路线效率。 |
| 爬塔 | 单队 DPS 深度和垂直推进能力。 | 主 C 强度、爆发、专武、星盘、升星、装备毕业度。 |
| 大秘境 | box breadth、环境适配和终局深度。 | 多角色养成、映射装备、旅券、专武、属性适配。 |
| BOSS挑战 | boss handling 和阶段性难度门槛。 | 爆发、生存、机制处理、专武/星盘/旅券效果。 |

## 判断方法

当一个成长系统发生变化时，按以下顺序判断影响：

1. 它提高的是单角色强度，还是账号广度。
2. 它提高的是基础面板，还是机制、节奏或构筑。
3. 它是稳定成长，还是随机毕业追求。
4. 它主要改变一次性通关、重复 farm、赛季竞争，还是终局挑战。
5. 它是否会让某个验证玩法被绕过或奖励效率异常。

## 常见关系

- `角色等级 / 突破` 更容易影响主线、资源副本和早期 BOSS。
- `星盘 / 升星 / 专武` 更容易影响角色机制、爆发窗口和 boss/爬塔表现。
- `旅券` 更容易影响中后期稳定战力和 boss/爬塔/大秘境表现。
- `映射装备` 更容易影响后期 farm、终局深度、大秘境和无限挑战。
- `炮台/基地相关成长` 优先看 PVPVE，不要外推到所有玩法。
