# 战斗转化字段与属性口径

本文件只记录成长到战斗表现的判断口径。精确公式和数值计算继续读取 `senior-game-economy/references/vgame-battle-system.md` 与 `game-numerical-analysis`。

## 属性类别

| 类别 | 示例 | 主要意义 |
|---|---|---|
| 基础属性 | 生命、物理攻击、魔法攻击、防御力 | 决定角色战斗底盘。 |
| 战斗属性 | 暴击、暴伤、命中、闪避、无视防御、伤害加成、伤害减免、治疗、护盾、格挡、冷却、能量等 | 决定战斗风格和专项表现。 |
| 百分比成长属性 | 生命%、物攻%、魔攻%、防御%、攻击速度%、能量回复% | 战斗外成长阶段放大基础属性。 |
| 等级属性 | 暴击等级、暴伤等级、能量回复等级、冷却等级、格挡等级 | 通过转换参数变成对应战斗属性。 |

## 面板转化

基础面板按以下思路理解：

```text
裸体属性
-> 战斗外成长加总
-> 面板属性
-> 战斗内 buff 改写
-> 最终战斗表现
```

关键口径：

- 战斗外同类属性通常按加法合并。
- 面板生命/攻击/防御是固定值与百分比成长共同作用的结果。
- 战斗内 buff 和战斗外成长分层计算。
- float 类战斗属性通常按加法合并。

## 伤害与生存链路

| 目标 | 优先观察 |
|---|---|
| DPS | 攻击、伤害标签、暴击、暴伤、攻速、冷却、能量、技能/必杀机制。 |
| 爆发 | 必杀伤害、能量回复、冷却、技能窗口、天赋/专武机制。 |
| 生存 | 生命、防御、减伤、格挡、护盾、治疗、受治疗。 |
| 稳定性 | 命中、闪避、控制、护盾、恢复、跑酷容错。 |
| 长线耐久 | 生存属性、恢复、能量循环、队伍厚度。 |
| 环境适配 | 伤害类型、标签增伤、套装效果、映射装备词条、队伍宽度。 |

## Vgame 特别注意

- `普攻攻击速度` 表示每秒普攻次数，不是动作倍率。
- `能量回复速度` 会影响必杀释放节奏，且角色可通过能量球补充能量。
- 所有标签型伤害加成共用同一乘区，先求和再进入最终伤害。
- 格挡有概率和减伤两个层面，当前还涉及格挡等级、格挡强度和破击。
- 真实伤害绕过普通伤害公式与护盾，但仍需确认最终乘区是否存在例外。
