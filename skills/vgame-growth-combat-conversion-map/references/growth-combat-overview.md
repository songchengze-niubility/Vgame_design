# 成长到战斗转化总览

## 总链路

Vgame 的成长转化链可以按以下顺序理解：

```text
主线推进 / 系统开放
-> 稳定成长供给和外循环补给
-> 角色、专武、旅券、映射装备等成长模块
-> 面板属性、技能机制、战斗内节奏和构筑深度
-> 正式玩法验证
```

Vgame 不是传统站桩卡牌。成长价值不仅体现在战力数值，还体现在：

- 跑酷战斗中的持续输出能力
- boss 处理能力
- 生存和容错
- 技能和必杀释放节奏
- 不同环境、怪物和玩法机制下的适配能力
- 长线验证玩法中的队伍深度

## 成长层级

| 层级 | 主要系统 | 职责 |
|---|---|---|
| 角色本体成长 | 升级、突破、星盘、升星 | 打底属性、节奏门槛、角色差异化、高价值节点。 |
| 角色机制成长 | 星盘、升星、专武 | 技能、天赋、被动和角色机制深化。 |
| 稳定装备成长 | 旅券 | 中后期稳定装备成长、套装效果和可预期毕业路径。 |
| 终局随机成长 | 映射装备、宝石、铭文 | 后期随机坑深、毕业追求、长期日活目标。 |
| 玩法专属支撑 | 炮台、基地相关成长等 | 主要影响 PVPVE 等特定验证玩法。 |

## 战斗转化层级

| 成长输出 | 战斗侧表现 |
|---|---|
| 基础属性 | 生命、物攻、魔攻、防御，决定底盘。 |
| 基础属性% | 放大基础底盘，和固定值成长形成乘加关系。 |
| 二级属性 | 暴击、暴伤、能量、冷却、格挡、伤害标签等，影响战斗风格。 |
| 技能/天赋/被动 | 改变角色机制、输出窗口、控制、生存或辅助能力。 |
| 套装/特殊效果 | 形成构筑差异和玩法适配。 |
| 随机词条 | 提供长期追求和属性最优解空间。 |

## 使用边界

- 问“这个成长系统最终影响什么”时，用本 skill。
- 问“这个资源从哪来、花到哪去”时，用 `vgame-economy-source-map`。
- 问“这个奖励包具体给什么”时，用 `vgame-reward-drop-sync`。
- 问“这个系统什么时候开”时，用 `vgame-level-progression-map`。
- 问“数值是否合理、曲线是否健康”时，用 `game-numerical-analysis` 或 `senior-game-economy`。
