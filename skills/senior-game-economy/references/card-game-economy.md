# 卡牌经济

当任务涉及卡牌游戏奖励、成长节奏、养成系统、商店定价，或任何可能影响抽卡与长线经济的设计时，读取本文件。

## 核心判断

- 不要让边缘玩法变成核心角色养成的主供给。
- 保护抽卡、商店和时间门槛成长线，不被旁路系统替代。
- 更鼓励阵容深度与养成广度，而不是单队绝对碾压。
- 卡牌游戏属于高密度经济系统，一个点超发，往往会联动扭曲多个系统。

## 资源分级

### 可高频投放资源

- 金币或其他软货币
- 角色经验
- 低中阶突破材料
- 替代性较弱、兑换压力较低的通用材料

### 需控制节奏的资源

- 角色抽卡券碎片
- 专武抽卡券碎片
- 定向培养材料
- 有价值但不是绝对主瓶颈的中高阶成长道具

### 高敏感资源

- 完整角色抽卡券
- 高稀有度角色万能碎片
- 核心觉醒、升星、升阶关键道具
- 高价值付费货币
- 直接决定 PVP 上限的资源

默认规则：凡是能替代抽卡、压穿后期瓶颈、明显冲击商业化的资源，都视为高敏感资源。

## 奖励设计结构

建议使用分层结构：

1. 基础层：高频、稳定、可预期。
2. 节点层：每隔几步给一次明显反馈。
3. 阶段层：每隔更大区间给一次可记忆奖励。
4. 荣誉层：后期逐步把价值从战力转向身份与证明。

## 节奏规则

- 前期：让玩家明显感到成长。
- 中期：单位奖励效率保持稳定，不继续陡增。
- 后期：成长价值放缓，动力更多转向荣誉、竞争和收藏。

## 反模式

- 高敏感资源稳定无限产出。
- 边缘玩法效率高过主线成长玩法。
- 奖励只做数值增加，没有节点记忆点。
- 只服务顶级玩家，忽视中坚玩家的目标感。
- 长线追逐只看单队战力，不鼓励阵容深度。

## 评审清单

- 这个系统会不会和抽卡形成替代关系？
- 这个系统会不会冲击商店包和通行证价值？
- 这个系统是否只奖励最强阵容，而没有鼓励阵容深度？
- 中坚玩家是否仍有可达成的阶段目标？
- 后段玩家拿到的是身份感，还是失控的战力收益？
