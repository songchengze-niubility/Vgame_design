# Vgame 术语总览

基于历史 `Vgame术语表.xlsx` 整理。当前工作区未发现该 Excel 原文件时，以本文档作为 Vgame 术语口径。

本文档用于统一 Vgame 在战斗、养成、经济、商店、奖励相关场景中的术语口径，作为后续讨论与落地时的优先参照。

## 1. 使用原则

- 优先使用“标准名”沟通。
- 若历史文档、经济表、商店表使用别名，则按本文档映射回标准名理解。
- `道具术语表` 中的“正式可配置道具”表示配置中有具体 id。
- `指代类道具` 表示用于策划表达、投放池或价值口径，不一定能直接配置投放。
- `非正式道具` 表示当前游戏内未实现，但可能在策划文档里作为概念存在。

## 2. 品质体系

### 2.1 颜色品质

| 颜色口径 | 品质名 | 英文名 | 品质值 |
|---|---|---|---|
| 蓝色、蓝 | 普通 | Blue | 2 |
| 紫色、紫 | 稀有 | Purple | 3 |
| 金色、金、橙色 | 史诗 | Orange | 4 |
| 红色、红 | 传说 | Red | 5 |

### 2.2 实体稀有度映射

- `普通 = B = R`
- `稀有 = A = SR`
- `史诗 = S = SSR`
- `传说 = SS = UR`

## 3. 养成体系术语

### 3.1 角色成长

- `主角等级`：也可称 `玩家等级`、`队伍等级`，是账号成长等级。
- `角色升级`：角色等级成长。
- `角色突破`：也可称 `角色升阶`，用于突破等级上限。
- `角色升星`：通过重复获取角色或碎片提升星级。
- `角色能效`：也可称 `角色星盘`、`技能树`，是角色技能树型成长系统。
- `能效点数`：也可称 `星盘点`、`星盘点数`、`能效点`，是角色能效系统的基础消耗点数。
- `免费星盘点`：角色默认可获得的点数。
- `额外星盘点`：稀有追加点数资源。
- `天赋`：角色成长中解锁或强化的额外被动/机制效果。
- `品阶系数`：裸体属性计算中的阶段放大系数。
- `角色裸体数值`：不包含装备等外部附加时，由角色自身成长系统提供的基础属性总和。

### 3.2 专武成长

- `专武`：角色可携带的一件专属武器或挂件。
- `专武升级`：专武等级成长。
- `专武突破`：也可称 `专武升阶`。
- `专武升星`：也可称 `专武精炼`，通过重复获取专武或万能材料提升星级。
- 当前口径中，`升阶` 与 `突破` 同义。

### 3.3 装备与挂载

- `旅券`：也可称 `锦囊`、`圣遗物`，角色身上的 6 部位装备系统。
- `旅券升级`：提升旅券等级。
- `旅券主属性`：不同部位从对应随机池中生成，不是所有部位完全固定。
- `旅券副属性`：不同部位共用同一副属性池。
- `映射装备`：也可称 `主角装备`，账号后期的全局装备养成系统。
- `宝石`：镶嵌在映射装备孔位上的养成道具，可合成更高等级。
- `铭文`：镶嵌在映射装备孔位上的养成道具，可形成套装效果，当前不继续合成。
- `孔位`：也可称 `孔槽`，映射装备上的镶嵌位置。
- `主属性`：也可称 `主词条`。
- `副属性`：也可称 `副词条`。
- `套装属性`：当前旅券 `2件套` 提供的属性。
- `套装效果`：当前旅券 `4件套` 提供的效果。
- `小被动`：附着在旅券套装或单套装备上的小型被动效果。
- `角色挂件`：可泛指挂载在角色身上的附属装备或机制载体，如旅券、专武。

### 3.4 商业化与招募

- `角色抽卡`：也可称 `角色招募`。
- `专武抽卡`：也可称 `专武招募`。
- `角色招募券`：也可称 `角色抽卡券`。
- `专武招募券`：也可称 `专武抽卡券`。
- `抽卡大代币`：高价值抽卡兑换货币。
- `抽卡小代币`：次级抽卡兑换货币。

### 3.5 品质实体

- `S角色`：也可称 `SSR角色`。
- `A角色`：也可称 `SR角色`。
- `B角色`：也可称 `R角色`。
- `S专武`：也可称 `SSR专武`。
- `A专武`：也可称 `SR专武`。
- `B专武`：也可称 `R专武`。

## 4. 战斗体系术语

### 4.1 属性层级

- `基础属性`：生命、物理攻击、魔法攻击、防御力。
- `战斗属性`：暴击、命中、闪避、伤害加成、格挡、能量等直接参与战斗结算的属性。
- `百分比加成属性`：也可统称 `基础属性%`，如生命%、攻击%、防御%、攻速%、能量回复%。
- `等级属性`：通过固定转化公式换算为实际战斗加成的属性，如暴击等级、暴伤等级、能量回复等级、技能冷却等级。

### 4.2 核心基础属性

- `生命`
- `物理攻击`：也可称 `物攻`
- `魔法攻击`：也可称 `魔攻`
- `防御力`：也可称 `防御`

### 4.3 资源机制

- `能量上限`
- `能量初始值`：也可称 `初始能量`
- `能量回复速度`：也可称 `能量回复`
- `能量球`：关卡内拾取后回复额外能量

### 4.4 常见战斗属性

- `暴击`：也可称 `暴击率`
- `暴击伤害`：也可称 `暴伤`
- `命中`
- `闪避`
- `无视防御`
- `普攻攻击速度`：也可称 `攻击速度`、`攻速`，表示每秒普攻次数
- `伤害减免`：也可称 `减伤`
- `受到伤害增加`：也可称 `易伤`
- `格挡`：也可称 `格挡率`
- `格挡等级`
- `格挡强度`
- `破击`：也可称 `破格挡`
- `冷却缩减`：也可称 `CD缩减`
- `效果命中`、`效果抵抗`：当前版本暂时不使用，可忽略

### 4.5 伤害乘区与标签

- `伤害加成`：也可称 `增伤`
- `物理伤害加成`
- `魔法伤害加成`
- `普攻伤害加成`
- `技能伤害加成`
- `必杀伤害加成`
- `单体伤害加成`
- `穿透伤害加成`
- `范围伤害加成`：也可称 `AOE伤害加成`
- `持续伤害加成`
- `流血伤害加成`
- `灼烧伤害加成`
- `中毒伤害加成`
- `玩法伤害加成`
- `玩法伤害减免`
- `同一乘区`：也可称 `同乘区`，多个标签型伤害加成先相加，再统一参与乘算
- `最终乘区`：基础伤害、暴击、格挡之后的最终乘算阶段

### 4.6 治疗与护盾

- `治疗`
- `恢复`
- `治疗加成`
- `受治疗加成`
- `施加护盾加成`
- `受到护盾加成`
- `释放盾牌加成`：作用于召唤出的盾牌生命，和护盾不同
- `护盾`

### 4.7 伤害类型与流程

- `真实伤害`：不进普通伤害公式，不受护盾影响，直接扣生命，但参与最终乘区
- `物理伤害`
- `魔法伤害`
- `伤害基数`
- `防御有效值`
- `命中保底`
- `基础伤害保底`
- `最终伤害保底`
- `基础格挡减伤`
- `命中判定`
- `暴击判定`
- `格挡判定`

### 4.8 持续效果

- `持续伤害`
- `流血`
- `灼烧`
- `中毒`

当前口径中，上述持续效果独立结算、互不干扰。

## 5. 道具体系术语

### 5.1 道具类型

- `正式可配置道具`：配置表中使用，有具体 id。
- `指代类道具`：用于投放池、价值口径或策划表达，不一定直接配置。
- `非正式道具`：当前游戏内未实现的概念型道具。

### 5.2 通用资源

- `猎鹰金币`：也可称 `金币`
- `元晶`：也可称 `钻石`
- `储能`：也可称 `体力`
- `行旅帧片`：也可称 `主角经验`、`玩家经验`
- `纳米墨核`：也可称 `旅券经验`
- `梦境碎片`：也可称 `爬塔代币`
- `专武商店代币`：也可称 `专武代币`
- `抽卡大代币`：也可称 `角色抽卡大代币`
- `抽卡小代币`：也可称 `角色抽卡小代币`

### 5.3 招募与成长道具

- `车票`：也可称 `角色抽卡券`、`角色招募券`
- `专武票`：也可称 `专武抽卡券`、`专武招募券`
- `遗迹逸能`：也可称 `角色经验低500`
- `浸染法则`：也可称 `角色经验中2000`
- `元能刻印`：也可称 `角色经验高10000`
- `星尘屑`：也可称 `专武经验低1000`
- `脉流结晶`：也可称 `专武经验中4000`
- `界碑星髓`：也可称 `专武经验高20000`
- `万象创痕`：也可称 `额外星盘点`、`万能星盘点`、`额外能效点`
- `元能核心`：也可称 `体力药`、`体力药60`

### 5.4 角色与专武材料

- `角色升阶材料低攻击/防御/辅助`
- `角色升阶材料中攻击/防御/辅助`
- `角色升阶材料高攻击/防御/辅助`
- `专武升阶材料低攻击/防御/辅助`
- `专武升阶材料中攻击/防御/辅助`
- `专武升阶材料高攻击/防御/辅助`
- `角色万能升阶材料`：当前归类为非正式道具，也可见历史口径 `角色万能突破材料`
- `专武万能突破材料`：当前归类为非正式道具
- `群星残契`：也可称 `B专武通用升星材料`、`B专武万能材料`
- `双生歧轨`：也可称 `A专武通用升星材料`、`A专武万能材料`
- `独一悖论`：也可称 `S专武通用升星材料`、`S专武万能材料`

### 5.5 星盘与能效材料

- `α型生长因子-1` 等：对应 `星盘低级/中级/高级/顶级道具攻击1`
- `元能辐蔽...` 系列：对应 `星盘低级/中级/高级/顶级道具防御1`
- `异质...` 系列：对应 `星盘低级/中级/高级/顶级道具辅助1`
- `抗疲累针剂/强效抑制剂/内啡肽缓释剂/强啡肽注射剂`：对应 `攻击2`
- `纳米再生粒子/再生反应堆/重构阵列/虚无拟态`：对应 `防御2`
- `欲念钥匙/灵觉钥匙/藏识钥匙/魂质钥匙`：对应 `辅助2`
- `星盘低级道具`、`星盘中级道具`、`星盘高级道具`、`星盘顶级道具`：均为指代类道具

### 5.6 映射装备与旅券相关道具

- `装备经验`：也可称 `主角装备经验`、`映射装备经验`
- `映射装备高级材料`：与 `传说装备兑换材料` 为严格同物同 id 的正式别名
- `映射装备低级材料`：与 `史诗装备兑换材料` 为严格同物同 id 的正式别名
- `映射通行券`：也可称 `映射装备副本钥匙`、`映射装备副本次数`
- `蓝色/紫色/金色/红色精密打孔器`：也可称同色 `打孔器`
- `蓝色/紫色/金色/红色鉴定卷轴`
- `1级-5级宝石`
- `1级-3级铭文`
- `红旅券`：也可称 `传说旅券`
- `金旅券`：也可称 `史诗旅券`
- `紫旅券`：也可称 `稀有旅券`
- `蓝旅券`：也可称 `普通旅券`
- `映射装备蓝色/紫色/金色/红色`：也可称对应 `主角装备颜色` 和对应品质映射装备

### 5.7 其他指代类道具

- `头像框`
- `头像`
- `时装`
- `角色经验`
- `专武经验`
- `S角色 / A角色 / B角色`
- `S专武 / A专武 / B专武`
- `SSR装备 / SR装备 / R装备`
- `SSR装备随机 / SSR装备自选`
- `SSR装备突破材料 / SR装备突破材料 / R装备突破材料`

以上条目当前保留为指代类道具，不转为正式可配置道具。

## 6. 当前约定

- `升阶` 与 `突破` 同义。
- `映射装备高级材料/低级材料` 与 `传说装备兑换材料/史诗装备兑换材料` 是严格同物同 id 的正式别名。
- `头像框`、`头像`、`时装` 维持为指代类道具。
- 讨论 Vgame 的战斗、养成、经济、商店、奖励问题时，优先按本文档对齐术语，而不是按通用行业口径猜测。
