# Vgame 养成结构总览

基于以下资料整理：

- [Vgame养成数值.xlsx](${VGAME_ROOT}/策划/数值组/2026-03/Vgame养成数值.xlsx)
- [Vgame基本战斗框架.xlsx](${VGAME_ROOT}/策划/数值组/2026-03/Vgame基本战斗框架.xlsx)
- [vgame-terminology.md](${VGAME_SKILL_ROOT}/senior-game-economy/references/vgame-terminology.md)
- [vgame-battle-system.md](${VGAME_SKILL_ROOT}/senior-game-economy/references/vgame-battle-system.md)

本文档用于总结 Vgame 当前版本的养成体系结构、模块职责、属性投放分工、角色差异化方式，以及各模块在玩家成长周期中的位置。

## 1. 总体结论

Vgame 的养成结构不是单线堆数值，而是一个前中后期逐步展开的组合体系：

- 前期靠 `角色本体成长` 建立基础战力
- 中期加入 `专武` 和 `旅券`，补足角色机制和稳定装备成长
- 后期用 `映射装备` 承接随机深度、毕业追求和长期坑深
- `角色星盘/能效` 与 `角色升星` 贯穿中长期，负责差异化成长和高价值提升

如果压缩成一句话：

`角色本体成长打底，专武承载机制，旅券提供稳定装备成长，映射装备承接终局随机深度，星盘和升星负责角色差异化与中长线拉升。`

## 2. 养成体系分层

从结构上看，Vgame 的养成可以拆成 5 层：

1. `角色裸体成长层`
2. `专武成长层`
3. `旅券成长层`
4. `映射装备终局层`
5. `角色差异化配置层`

它们共同作用于战斗属性，但职责并不相同。

## 3. 角色裸体成长层

角色裸体成长是所有养成的底盘，不包含旅券、映射装备等外部挂载系统。

根据 [Vgame养成数值.xlsx](${VGAME_ROOT}/策划/数值组/2026-03/Vgame养成数值.xlsx) 与 [Vgame基本战斗框架.xlsx](${VGAME_ROOT}/策划/数值组/2026-03/Vgame基本战斗框架.xlsx) 的口径，当前裸体成长主要包括：

- `初始属性`
- `角色升级`
- `角色突破`
- `角色星盘 / 角色能效`
- `角色升星`

### 3.1 初始属性

作用：

- 决定角色出生底盘
- 为职业、品质、伤害类型差异提供基础空间

当前标准人初始属性口径：

- 生命：`500`
- 攻击：`150`
- 防御：`100`

### 3.2 角色升级

作用：

- 前中期最稳定、最普惠的基础属性来源
- 主要提供线性成长

当前设计：

- 等级上限 `60`
- 单级成长比例 `0.08`
- 主要投放 `基础属性`

结构意义：

- 玩家最早接触、最容易理解
- 为早期战力建立最直接的正反馈

### 3.3 角色突破

作用：

- 控制角色成长节奏和等级门槛
- 绑定其他模块开放节奏，例如技能、星盘节点等

当前正式设计：

- 突破本身不直接提供额外属性成长
- 主要承担“卡等级上限”的作用

说明：

- 这与很多游戏“突破直接加属性”的模式不同
- 在 Vgame 当前口径中，突破更偏节奏控制，而不是单纯数值投放

### 3.4 角色星盘 / 角色能效

作用：

- 承担角色中长期成长
- 承担角色偏向性塑造
- 承担一部分技能/被动成长

术语口径：

- `角色能效 = 角色星盘 = 技能树`
- `能效点数 = 星盘点 = 星盘点数 = 能效点`

当前结构：

- 总点位 `27`
- 免费点位 `21`
- 额外点位 `6`
- 属性点位 `8`
- 技能点位 `19`

主要投放：

- 一部分 `基础属性`
- 一部分 `基础属性%`
- `普通二级属性`
- 技能升级或被动成长

系统意义：

- 是角色个性化成长的重要来源
- 和角色定位、技能设计深度绑定

### 3.5 角色升星

作用：

- 负责高价值成长
- 负责天赋解锁
- 负责普通/稀有二级属性的高强度投放

当前设计特征：

- 共 6 星
- 其中 4 个节点投放天赋或被动提升
- 2 个节点投放属性%
- 整体升星带来的战中提升预期可达 `1.8 - 2 倍`

当前节点设计：

- `1星`：天赋1
- `2星`：天赋2
- `3星`：全属性15% + 普通二级属性20%
- `4星`：天赋3
- `5星`：全属性15% + 稀有二级属性20%
- `6星`：天赋4

系统意义：

- 明显偏高价值成长和商业化深度
- 不是日常平滑成长，而是关键节点型成长

## 4. 专武成长层

专武在 Vgame 中不是纯数值装备，更像：

- 角色机制的延伸
- 一部分基础属性来源
- 一部分偏向属性来源

### 4.1 专武职责

专武主要承担三件事：

- 承载一部分角色技能机制
- 提供一部分 `基础属性`
- 提供一部分 `基础属性% + 普通二级属性`

### 4.2 专武成长结构

当前结构分为：

- `专武升级`
- `专武突破`
- `专武升星`

职责拆分：

- `专武升级`：投基础属性
- `专武突破`：投偏向属性
- `专武升星`：投技能机制成长

### 4.3 当前核心参数

- 专武上限等级：`60`
- 基础属性占比：`15%`
- 每级属性成长：`1.42%`

### 4.4 系统意义

专武和角色本体的关系不是平行外设，而是深度绑定：

- 角色决定专武需求
- 专武反过来强化角色机制

因此专武在项目中承担的是：

`机制挂件 + 中长期成长线`

## 5. 旅券成长层

旅券是角色中后期的稳定装备系统，属于“可预期成长线”。

### 5.1 系统定位

当前口径：

- `旅券 = 锦囊 = 圣遗物`
- 一共 `6` 个部位

设计目标：

- 承担一部分中后期属性追求
- 承担套装属性和小被动
- 相比传统随机装备，更强调可毕业预期

### 5.2 套装结构

当前设计：

- `4+2` 套装结构
- `2件套` 提供属性
- `4件套` 提供效果
- 每套旅券带一个 `小被动`

预期：

- 6件总战斗提升期望约 `30%`

### 5.3 主要投放内容

旅券主要投放：

- 一部分 `基础属性`
- 主属性
- 副属性
- 一部分 `普通二级属性`
- 套装属性
- 套装效果

当前预计基础属性投放：

- 生命：`3000`
- 攻击：`900`
- 防御：`600`

当前预计二级属性投放：

- 总偏向值约 `420%`

### 5.4 主副属性结构

旅券主副属性权重：

- 主属性权重：`32%`
- 副属性权重：`68%`

主属性二级属性量：

- `134.4%`

副属性二级属性量：

- `285.6%`

### 5.5 品质与等级

旅券品质系数：

- 普通：`0.4`
- 稀有：`0.6`
- 史诗：`0.8`
- 传说：`1`

规划上限等级：

- `5`

### 5.6 主属性池

旅券 6 个部位的主属性分配并不相同：

- 前排部位偏基础属性
- 后排部位偏基础属性%、等级属性和伤害标签

例如会出现：

- 物理攻击%
- 魔法攻击%
- 生命%
- 防御力%
- 必杀回复速度等级
- 冷却缩减等级
- 暴击等级
- 暴击伤害等级
- 格挡等级
- 普攻/技能/必杀伤害加成
- 治疗率

### 5.7 系统意义

旅券的核心特征不是极致随机，而是：

- 规律性更强
- 成长更平滑
- 毕业路径更清晰

所以它在整个养成结构中的位置是：

`中后期稳定装备成长线`

### 5.8 旅券合成口径

旅券合成的定位是：`红色旅券补坑系统`。

当前规则口径：

- 旅券合成只产出 `红色旅券`，不产出蓝、紫、金品质。
- 合成出的旅券等级为 `0`。
- 合成不会继承材料旅券的等级、主属性、副属性或强化结果。
- 主属性如果可指定，必须符合该部位允许出现的主属性池。
- 副属性永远不能指定，始终随机，用来保留红色旅券的长期毕业追求。

三档合成规则：

| 合成类型 | 可指定内容 | 随机内容 | 消耗 |
| --- | --- | --- | --- |
| 低级合成 | 指定套装 | 部位随机、主属性随机、副属性随机 | 低级合成材料 |
| 高级合成 | 指定套装 + 指定部位 | 主属性随机、副属性随机 | 高级合成材料 |
| 特级合成 | 指定套装 + 指定部位 + 指定主属性 | 副属性随机 | 高级合成材料 + 特级合成材料 |

设计边界：

- 低级合成解决 `我要这套红旅券`。
- 高级合成解决 `我要这套的这个部位`。
- 特级合成解决 `我要这套、这个部位、这个主属性`。
- 合成只解决红旅券成型补坑，不直接解决副属性毕业。

## 6. 映射装备终局层

映射装备是 Vgame 的后期终局成长系统，也是主要的坑深来源。

术语口径：

- `映射装备 = 主角装备`

### 6.1 系统定位

设计页明确给出的定位是：

- 玩家终局养成系统
- 用随机词条带来极强坑深
- 承载后期玩家日活和长期毕业追求

### 6.2 主要投放内容

映射装备主要投放：

- `基础属性`
- `基础属性%`
- `普通二级属性`

当前预计投放：

- 基础属性：生命 `4000` / 攻击 `1200` / 防御 `800`
- 基础属性%：生命 `240%` / 攻击 `240%` / 防御 `240%`
- 偏向二级属性：`320%`

### 6.3 系统结构

映射装备由以下部分组成：

- 主属性
- 副词条
- 孔位
- 宝石
- 铭文

特点：

- 主属性固定
- 副词条强随机
- 孔位数量随机
- 宝石可合成
- 铭文当前不可继续合成

### 6.4 部位设计

当前主要部位包括：

- 武器
- 头盔
- 铠甲
- 手套
- 腰带
- 鞋子
- 装置
- 饰品

不同部位绑定不同主属性池，例如：

- 武器偏 `物攻 / 法攻 / 暴击等级`
- 头盔偏 `生命 / 物攻%`
- 铠甲偏 `防御 / 防御%`
- 手套偏 `防御 / 魔攻%`
- 腰带偏 `生命 / 生命%`
- 鞋子偏 `物攻 / 冷却缩减等级`
- 装置偏 `法攻 / 能量回复等级`
- 饰品偏 `暴击等级 / 暴击伤害等级`

### 6.5 品质结构

当前稀有度部分：

- 蓝：等级上限 `30`
- 紫：等级上限 `40`
- 金：等级上限 `50`
- 红：等级上限 `60`

品质系数：

- 蓝：`0.3`
- 紫：`0.5`
- 金：`0.75`
- 红：`1`

### 6.6 副词条结构

副词条当前覆盖大量属性，包括：

- 物攻%
- 魔攻%
- 防御%
- 生命%
- 技能冷却等级
- 能量回复等级
- 暴击等级
- 暴击伤害等级
- 物攻
- 魔攻
- 防御
- 生命
- 物理伤害加成
- 魔法伤害加成

并且每个品质定义了：

- 数值上限
- 数值下限
- 最小单位
- 权重
- 评分基础值

### 6.7 宝石与铭文

映射装备上的孔位可镶嵌：

- `宝石`
- `铭文`

其中：

- `宝石` 更偏进度型成长，可合成
- `铭文` 更偏效果和套装深度，当前不继续合成

### 6.8 系统意义

映射装备在整个养成结构中的角色是：

`后期终局随机深度系统`

它不是简单补数值，而是承载：

- 长期毕业追求
- 日常活跃目标
- 属性最优解追求
- 宝石/铭文组合深度

### 6.9 红装洗练与超限口径

映射装备红装的当前设计口径是：`固定构筑模板 + 数值随机 + 掉落惊喜`。

红装基础结构：

- 红装固定拥有 `4条属性词条类型 + 1条技能词条`。
- 4条属性词条的属性类型由红装模板决定，不随机。
- 技能词条由红装模板决定，不参与普通洗练。
- 随机主要发生在属性数值上。

洗练规则：

- 洗练只针对红装的 4 条固定属性词条数值生效。
- 洗练不会改变属性类型、技能词条、装备模板或超限资格。
- 每次洗练后，玩家可在 `原结果` 和 `新结果` 之间二选一。
- 洗练支持锁定词条，锁定词条不参与本次随机。
- 锁定数量越多，单次洗练消耗越高。
- 普通红装不能通过洗练获得超限词条。

超限规则：

- 超限只在红装掉落或生成时判定，不能通过普通洗练获得。
- 如果触发超限，会从该红装固定的 4 条属性中随机选 1 条进入超限状态。
- 超限不会新增属性类型，只让已有属性突破正常数值上限。
- 一件红装最多只有 1 条超限词条。
- 带超限的红装后续洗练时，超限资格永久保留。
- 超限词条未锁定时，洗练后仍在超限区间内重新随机。
- 超限词条已锁定时，保留当前超限数值。

设计边界：

- 洗练负责让目标红装数值可培养，降低成型坑深。
- 超限负责保留红装掉落/生成时的天生极品惊喜。
- 超限不是 build 门槛；没有超限的目标红装仍应完整可用。

## 7. 属性类型分工

整套养成体系投放的属性，本质上可以拆成 4 类：

1. `基础属性`
2. `基础属性%`
3. `技能等级 / 技能机制成长`
4. `二级属性`

### 7.1 基础属性

主要由以下模块承担：

- 角色升级
- 角色突破（当前不直接投额外成长，但仍在结构上属于基础成长线）
- 角色星盘
- 专武升级
- 旅券
- 映射装备主属性

### 7.2 基础属性%

主要由以下模块承担：

- 角色星盘
- 角色升星
- 专武突破
- 旅券部分主属性
- 映射装备主属性与副词条

### 7.3 技能等级 / 技能成长

主要由以下模块承担：

- 角色星盘
- 角色升星（天赋）
- 专武
- 旅券小被动 / 套装效果
- 铭文

### 7.4 二级属性

主要由以下模块承担：

- 角色星盘
- 角色升星
- 专武突破
- 旅券
- 映射装备副词条

其中：

- `普通二级属性` 更普遍
- `稀有二级属性` 更偏高价值投放

## 8. 角色差异化结构

Vgame 并不是所有角色套同一套成长模板，而是通过“系数体系”来做差异化。

### 8.1 差异化来源

角色差异化当前主要由以下因素决定：

- `稀有度`
- `职业`
- `伤害类型`
- `角色微调系数`

### 8.2 职业模板

当前有多套职业模板，例如：

- 坦克（输出）
- 坦克（生命）
- 坦克（防御）
- 输出（射手）
- 输出（战士）
- 输出（法师）
- 辅助（均衡）
- 辅助（攻击）
- 辅助（生命）

### 8.3 伤害类型模板

当前伤害类型包括：

- 物理
- 魔法
- 混合

并且每种伤害类型会影响物攻/法攻的分配权重。

### 8.4 品质倍率

当前品质属性比：

- `SSR = 1`
- `SR = 0.8`
- `R = 0.7`

已确认口径：

- 该品质倍率只影响 `裸体模块 + 专武属性`
- 不直接代表所有养成模块都按同一倍率缩放

### 8.5 结果页

[配置页-不同武将系数配置](${VGAME_ROOT}/策划/数值组/2026-03/Vgame养成数值.xlsx) 会把上述规则落成每个角色的：

- 升级基础/成长属性
- 星盘基础属性
- 星盘偏向的 3 个二级属性
- 升星偏向属性
- 伤害类型标签

其中：

- `二级属性1-3星` 对应升星 3 星节点
- `二级属性2-5星` 对应升星 5 星节点

## 9. 玩家成长阶段体验

如果从玩家体验视角看，这套养成结构大致是：

### 9.1 前期

主驱动：

- 角色升级
- 角色突破
- 主角等级推进

体验特征：

- 成长直观
- 数值回报直接
- 系统理解成本低

### 9.2 中期

主驱动：

- 角色星盘
- 专武
- 旅券

体验特征：

- 开始出现角色分化
- 开始出现构筑和属性偏向
- 装备和机制成长同时接入

### 9.3 后期

主驱动：

- 映射装备
- 宝石
- 铭文
- 深度升星 / 专武升星

体验特征：

- 强随机长线追求
- 明显毕业空间
- 日活驱动更强

## 10. 各模块在项目中的职责总结

如果按一句话概括每个模块的职责：

- `角色升级`：前中期稳定基础成长
- `角色突破`：成长节奏门槛控制
- `角色星盘/能效`：中长期差异化成长与一部分技能成长
- `角色升星`：高价值成长、天赋和稀有属性投放
- `专武`：角色机制延伸 + 一部分成长线
- `旅券`：中后期稳定装备成长线
- `映射装备`：后期终局随机深度系统
- `宝石`：映射装备上的稳定进度成长
- `铭文`：映射装备上的效果与组合深度

## 11. 当前已确认口径

- `升阶` 与 `突破` 同义
- `角色突破` 当前正式设计不直接给额外成长值，主要承担门槛控制
- `角色能效 = 角色星盘 = 技能树`
- `不同武将系数` 只影响 `裸体模块 + 专武属性`
- `旅券` 当前正式口径已采用 `格挡等级`
- `映射装备` 原设计推导页为 `设计页-映射装备数值`
- 更便于理解和阅读的正式展开页为 `映射装备数值`
- `草稿页-映射装备配置` 为草稿页，可不作为正式依据
