# Release Status Registry

## 定位

本文档维护 Vgame 各功能/内容的发布状态。作为版本追踪的真实记录，需要从项目证据持续更新。

## 状态定义

| 状态 | 英文 | 说明 |
|---|---|---|
| 规划中 | PLAN | 已确认要做，未开始开发 |
| 开发中 | DEV | 正在开发，配置/代码进行中 |
| 内测中 | INTERNAL_TEST | 开发完成，内部测试验证 |
| 灰度中 | GRAY | 部分用户可见，观测指标 |
| 已上线 | LIVE | 全量玩家可用 |
| 维护中 | MAINTAIN | 不再迭代，只修bug |
| 已废弃 | DEPRECATED | 标记废弃，仍有残留配置/代码 |
| 已移除 | REMOVED | 配置和代码已清理 |
